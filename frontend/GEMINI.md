## Frontend Overview

The frontend is a React application responsible for the user interface. The main component is `TestFlowDesigner.js`, which allows users to design and visualize test flows.

### Key Files

*   `src/App.js`: The main application component.
*   `src/index.js`: The entry point for the React application.
*   `src/components/TestFlowDesigner.js`: The core component for designing test flows.
*   `public/index.html`: The main HTML file.

### Running the Frontend

To start the frontend development server, run the following command:

```bash
npm start
```
