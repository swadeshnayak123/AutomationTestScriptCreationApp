This file provides context and instructions for the Gemini AI agent.

## Executor Overview

The executor is a Node.js application responsible for running the tests designed in the frontend. It takes the test flow definition and executes the steps.

### Key Files

*   `src/index.js`: The main entry point for the executor.

### Running the Executor

To run the executor, use the following command:

```bash
node src/index.js
```
