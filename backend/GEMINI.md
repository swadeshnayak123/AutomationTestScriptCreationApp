## Backend Overview

The backend is a Node.js application that serves as the API for the frontend. It is responsible for handling business logic and data persistence.

### Key Files

*   `src/index.js`: The main entry point for the backend server.

### Running the Backend

To start the backend server, run the following command:

```bash
node src/index.js
```
