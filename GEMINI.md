This file provides context and instructions for the Gemini AI agent.

## Project Overview

This project is a full-stack application with the following components:

*   **frontend:** A React-based user interface.
*   **backend:** A Node.js server providing the API.
*   **executor:** A Node.js application for running tests.
*   **mongodb:** A local MongoDB instance for data storage.

## General Instructions

*   When working on a specific component, refer to the `GEMINI.md` file within that component's directory for more detailed instructions.
*   The database is a local MongoDB instance. The executable is located in `mongodb-win32-x86_64-windows-8.0.12/bin/mongod.exe`.
